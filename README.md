# How It Works
Fetch the recent activity of the specified GitHub user using the GitHub API.

# Usage
Provide the GitHub username as an argument when running the CLI.

Basic report:
```powershell
github-activity <username>
```
Detailed report:
```powershell
github-activity <username> <token>
```
